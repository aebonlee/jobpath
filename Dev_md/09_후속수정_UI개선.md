# JobPath — 후속 수정 및 UI 개선

**작성일:** 2026-04-06
**프로젝트:** jobpath (직업상담사 2급 시험 준비 플랫폼)

---

## 변경 이력

### 1. 푸터 간소화
- 4열 그리드(브랜드/학습메뉴/서비스/문의) → 중앙 정렬 1열 컴팩트 레이아웃
- 로고 + 주요 링크 5개(서비스소개, 요금제, 시험안내, 이용약관, 개인정보) + 연락처 1줄
- 패밀리사이트 드롭다운, 중복 메뉴 리스트 제거
- 다크모드/반응형 불필요 스타일 정리

### 2. PortOne SDK async 로딩
- `index.html`의 iamport.js 스크립트를 `async`로 변경
- `<div id="root">` 뒤로 이동하여 페이지 렌더링 블로킹 방지
- Checkout.tsx 깨진 한글 문자(`발생��습니다` → `발생했습니다`) 수정

### 3. 로그인 홈 히어로 상단 여백
- `.ph-hero` padding-top에 `var(--nav-height)` 보정 추가
- `padding: 48px 0 40px` → `calc(var(--nav-height) + 48px) 0 40px`
- 고정 Navbar(84px) 뒤에 히어로가 가려지던 문제 수정

### 4. D-day 시험일/접수일 구분
- `getDday()` 반환값에 `type` 필드 추가 (`'시험일'` | `'접수일'`)
- D-day 배지에 시험일/접수일 구분 태그 표시
- `.ph-dday-type` 스타일 추가 (노란색 배경 배지)

### 5. 홈페이지 구조 변경
- **기존**: 로그인 시 개인 대시보드만 표시 (메인 랜딩 숨김)
- **변경**: 로그인 시 상단 개인 대시보드 요약 + 하단 메인 랜딩 페이지 모두 표시
- 비로그인 시에는 메인 랜딩 페이지만 표시 (기존과 동일)
- `Home.tsx`: `{user ? <LoggedInHome /> : <LandingHome />}` → `{user && <LoggedInHome />}<LandingHome />`
