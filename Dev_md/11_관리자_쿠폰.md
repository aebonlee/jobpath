# JobPath - 관리자 대시보드 & 쿠폰 시스템

## 개요
- 관리자 대시보드 (회원 관리, 주문 관리, 쿠폰 관리)
- 홍보용 무료 1일 체험 쿠폰 발행/관리
- 일반 사용자용 쿠폰 등록 페이지

## 관리자 설정
- 관리자 이메일: `aebon@kakao.com`
- 설정 파일: `src/config/admin.js` (ADMIN_EMAILS 배열)
- AuthContext에서 isAdmin 상태 관리
- AdminGuard 컴포넌트로 /admin 라우트 보호

## 페이지 구성

### /admin - 관리자 대시보드
- 접근: AdminGuard (로그인 + 관리자만)
- 상단 통계 카드: 총 회원수, 총 주문수, 총 매출, 활성 쿠폰수
- 탭 구조:
  1. **회원 관리**: forjob_orders에서 유니크 이메일 추출, 주문수/최근결제일/구독상태
  2. **주문 관리**: 전체 주문 목록 (주문번호, 이메일, 플랜, 금액, 상태, 결제일)
  3. **쿠폰 관리**: 쿠폰 발행/목록/비활성화/일괄복사

### /coupon - 쿠폰 등록
- 접근: AuthGuard (로그인 필요)
- 쿠폰 코드 입력 → 검증 → 1일 이용권 자동 활성화
- 검증 항목: is_active, used_count < max_uses, expires_at, 중복 사용 방지

## Supabase 테이블

### forjob_coupons
```sql
CREATE TABLE forjob_coupons (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  code TEXT UNIQUE NOT NULL,
  plan_type TEXT NOT NULL DEFAULT '1day_trial',
  days INTEGER NOT NULL DEFAULT 1,
  max_uses INTEGER NOT NULL DEFAULT 1,
  used_count INTEGER NOT NULL DEFAULT 0,
  is_active BOOLEAN NOT NULL DEFAULT true,
  created_by TEXT,
  created_at TIMESTAMPTZ DEFAULT now(),
  expires_at TIMESTAMPTZ
);
```

### forjob_coupon_redemptions
```sql
CREATE TABLE forjob_coupon_redemptions (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  coupon_id UUID REFERENCES forjob_coupons(id),
  user_id UUID NOT NULL,
  user_email TEXT,
  redeemed_at TIMESTAMPTZ DEFAULT now()
);
```

### RLS 정책
```sql
ALTER TABLE forjob_coupons ENABLE ROW LEVEL SECURITY;
ALTER TABLE forjob_coupon_redemptions ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Anyone can read active coupons" ON forjob_coupons FOR SELECT USING (true);
CREATE POLICY "Auth users can insert redemptions" ON forjob_coupon_redemptions FOR INSERT WITH CHECK (auth.uid() = user_id);
CREATE POLICY "Users can read own redemptions" ON forjob_coupon_redemptions FOR SELECT USING (auth.uid() = user_id);
```
