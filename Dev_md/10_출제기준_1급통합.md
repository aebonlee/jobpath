## 변경 사항

### 1. 사이트 브랜딩 1급·2급 통합
- `site.js`: SITE.title, description에서 "2급" 제거 → "1급·2급" 통합 표현

- `Home.tsx`: 히어로 배지/타이틀 "1급·2급" 반영

### 2. site.js 설정 추가
- 기존 `SUBJECTS` → `SUBJECTS_2`로 명확화 (호환 alias 유지)
- `SUBJECTS_1` 신규: 1급 5과목 (직업심리 및 전직지원, 심층직업상담 및 슈퍼비전, 직업정보가공, 노동시장분석, 고용노동관계법규(II))
- `EXAM_CONFIG_1` 신규: 1급 필기 150분, 실기 150분

### 3. ExamInfo.tsx 탭 구조 전환
- 상단에 2급/1급 탭 전환 UI (`useState('level2')`)
- **2급 탭**: 기존 전체 섹션 + 출제기준(2025~2027) 섹션 추가
  - 필기 5과목 출제기준 카드 그리드
  - 실기 8개 영역 (신규 4개: 직업복귀상담, 직업훈련상담, 집단상담, 행정)
- **1급 탭**: 새로 작성
  - 시험 개요 (응시자격 경력 필요, 연 1회 3회차만)
  - 응시자격 상세 (2급 취득 후 3년 등)
  - 필기·실기 시험 정보
  - 필기 과목 5개
  - 출제기준 (필기 5과목 + 실기 4영역)
  - 시험일정 (3회차만)
  - 1급 학습 콘텐츠 향후 업데이트 안내

### 5. 홈 시험일정 3열 카드 레이아웃 (후속)
- 히어로 시험일정을 리스트형 → **3열 카드 박스** 디자인으로 전환
- 상위 3개 일정만 카드로 표시 (타입배지 + 라벨 + 날짜 + D-day 크게)
- 가장 가까운 일정은 황금색 강조
- `hero.css`: `.hero-schedule-cards` 3열 그리드, `.hero-schedule-card` 카드 스타일
- 반응형: 태블릿 3열 유지, 모바일(640px 이하) 1열 가로 배치

### 6. 과목 섹션 표시 버그 수정
- `Home.tsx` 과목 카드에 `fade-in-up` 클래스 적용되어 있었으나 Intersection Observer 미구현
- `opacity: 0` 상태로 렌더링되어 과목 섹션이 보이지 않는 버그 발생
- `fade-in-up` 클래스 제거로 해결

### 7. 시험일정 카드 전폭 디자인 개선
- 시험일정 카드를 `.hero-content`(max-width 700px) 밖으로 이동 → 컨테이너 전폭 활용
- 카드 크기 확대: padding 32px, D-day 28px→32px(nearest), 라벨 16px
- 카드 배경 글래스모피즘 강화, hover 시 6px 리프트 + 강한 그림자
- 모바일: 1열 가로 카드 (좌측 컬러 보더)

### 8. CTA~푸터 간 흰 배경 간격 제거
- `.footer`의 `margin-top: 80px` 때문에 CTA(어두운 배경)와 푸터 사이에 흰 배경 노출
- `.home-cta-section`에 `margin-bottom: -80px` 추가하여 상쇄
