# 10. 출제기준 추가 + 1급·2급 통합

## 작업일: 2026-04-06

## 변경 사항

### 1. 사이트 브랜딩 1급·2급 통합
- `site.js`: SITE.title, description에서 "2급" 제거 → "1급·2급" 통합 표현
- `index.html`: OG/Twitter 메타태그 "직업상담사 1급·2급" 반영
- `Home.tsx`: 히어로 배지/타이틀 "1급·2급" 반영

### 2. site.js 설정 추가
- 기존 `SUBJECTS` → `SUBJECTS_2`로 명확화 (호환 alias 유지)
- `SUBJECTS_1` 신규: 1급 5과목 (직업심리 및 전직지원, 심층직업상담 및 슈퍼비전, 직업정보가공, 노동시장분석, 고용노동관계법규(II))
- `EXAM_CONFIG_1` 신규: 1급 필기 150분, 실기 150분

### 3. ExamInfo.tsx 탭 구조 전환
- 상단에 2급/1급 탭 전환 UI (`useState('level2')`)
- **2급 탭**: 기존 전체 섹션 + 출제기준(2025~2027) 섹션 추가
  - 필기 5과목 출제기준 카드 그리드
  - 실기 8개 영역 (신규 4개: 직업복귀상담, 직업훈련상담, 집단상담, 행정)
- **1급 탭**: 새로 작성
  - 시험 개요 (응시자격 경력 필요, 연 1회 3회차만)
  - 응시자격 상세 (2급 취득 후 3년 등)
  - 필기·실기 시험 정보
  - 필기 과목 5개
  - 출제기준 (필기 5과목 + 실기 4영역)
  - 시험일정 (3회차만)
  - 1급 학습 콘텐츠 향후 업데이트 안내

### 4. CSS 추가
- `study.css`: `.info-level-tabs`, `.info-level-tab`, `.info-criteria-*` 스타일
- `dark-mode.css`: 탭/출제기준 다크모드
- `responsive.css`: 탭/출제기준 반응형 (태블릿 2열, 모바일 1열)
