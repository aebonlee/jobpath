# ForJob - 필기시험 디스플레이 설정 (2열 보기 + 크기 조절)

**작성일:** 2026-04-05
**프로젝트:** ForJob (직업상담사 2급 시험 준비 플랫폼)
**배포 URL:** https://forjob.dreamitbiz.com

---

## 개요

필기시험 문항 표시 방식을 개선했습니다:
1. **2열 보기** — 데스크탑에서 2문항을 좌우로 동시 표시
2. **크기 조절** — 작게/보통/크게 3단계 문항 크기 변경
3. **여백 개선** — 양쪽 프레임에 붙어보이던 문제 해결 (패딩 40px → 56px)
4. **설정 유지** — localStorage에 저장, 세션 간 유지

---

## 기능 상세

### 디스플레이 툴바

문항 영역 상단 우측에 설정 바 표시:

```
                              [보기] [1열|2열]  [크기] [작게|보통|크게]
```

- iOS 세그먼트 컨트롤 스타일
- 선택 상태: 흰색 배경 + 파란 텍스트 + 미세 그림자

### 1열 / 2열 보기

| 모드 | 동작 |
|------|------|
| **1열** (기본) | 문항 1개씩 표시, 1개씩 이동 |
| **2열** | 문항 2개 좌우 표시, 2개씩 이동, 페이지 표시 `1–2 / 100` |

- 2열 전환 시 `exam-body`에 `.dual-mode` 클래스 → `max-width: none`
- CSS Grid `1fr 1fr` 레이아웃, `gap: 20px`, `align-items: start`
- 마지막 페이지 홀수 문항이면 왼쪽만 표시
- **1280px 이하에서 2열 토글 숨김** (화면 폭 부족)

### 크기 조절

| 크기 | question-text | option-text | card padding | option padding |
|------|---------------|-------------|--------------|----------------|
| **작게** (sm) | 14px | 13px | 24px | 12px 14px |
| **보통** (md) | 16px | 15px | 32px | 16px 20px |
| **크게** (lg) | 18px | 17px | 40px | 20px 24px |

추가로 question-number 크기, options-list gap, question-header margin도 비례 조정.

### 여백 개선

`.exam-main` 좌우 패딩: `40px` → `56px` (+16px 각 방향)

## 기술 구현

### 상태 관리 (ExamMode.tsx)

```tsx
const [displayMode, setDisplayMode] = useState(() =>
  localStorage.getItem('forjob-exam-layout') || 'single'
);
const [questionSize, setQuestionSize] = useState(() =>
  localStorage.getItem('forjob-exam-size') || 'md'
);
```

- `displayMode`: `'single'` | `'dual'`
- `questionSize`: `'sm'` | `'md'` | `'lg'`
- `useEffect`로 localStorage 자동 저장

### 2열 페이지네이션 로직

```tsx
const step = displayMode === 'dual' ? 2 : 1;
const visibleQuestions = displayMode === 'dual'
  ? questions.slice(currentIndex, currentIndex + 2)
  : [currentQuestion];
const isLastPage = displayMode === 'dual'
  ? currentIndex + 2 >= questions.length
  : currentIndex >= questions.length - 1;
```

### CSS 클래스 구조

```
.exam-body.dual-mode
  └─ .exam-main.size-{sm|md|lg}
       ├─ .exam-display-toolbar
       │    ├─ .display-toggle-group.display-layout-toggle
       │    └─ .display-toggle-group (크기)
       ├─ .exam-questions-dual (2열 시)
       │    ├─ .question-card (좌)
       │    └─ .question-card (우)
       └─ .exam-nav-buttons
```

### 반응형 처리

| 브레이크포인트 | 변경 |
|----------------|------|
| ≤ 1279px | `.display-layout-toggle` 숨김, dual grid → 1열 |
| ≤ 767px | 툴바 gap 축소, 버튼 소형화 |
| ≤ 639px | 툴바 중앙 정렬, wrap, 라벨 숨김 |

## 사용자 경험

1. 넓은 모니터(1280px+)에서 2열로 전환하면 한 화면에 2문항을 비교하며 풀 수 있음
2. 크기 조절로 시력이나 선호에 따라 문항 가독성 조절 가능
3. 설정이 localStorage에 저장되어 다음 시험에서도 동일 설정 유지
4. 좌우 여백 증가로 문항 카드가 프레임에 붙지 않고 여유 있게 표시
