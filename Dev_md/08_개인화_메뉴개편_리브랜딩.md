# JobPath 사이트 전면 개편 — 개인화 + 메뉴 재구성 + 리브랜딩

> 작성일: 2026-04-06
> 작업: 사이트 전체 점검, 개인 맞춤형 학습 개인화, NAV 메뉴 재구성, ForJob → JobPath 리브랜딩

---

## 1. 리브랜딩: ForJob → JobPath

### 변경 범위
- 사이트명: `ForJob` → `JobPath`
- 도메인: `forjob.dreamitbiz.com` → `jobpath.dreamitbiz.com`
- Navbar 로고: `For / Job` → `Job / Path`
- Footer 로고: 동일 변경
- Favicon SVG: 텍스트 변경
- OG 이미지: 타이틀/URL 변경 후 재생성
- index.html: title, og:*, twitter:*, canonical 모두 변경
- 404.html: title 변경
- CNAME: 도메인 변경
- package.json: name, homepage 변경
- CSS 전체 파일 헤더 주석: ForJob → JobPath
- localStorage 키: `forjob-exam-*` → `jobpath-exam-*`

### 유지 항목
- Supabase 테이블 접두어 `forjob_` — 변경 없음 (DB 마이그레이션 불필요)

## 2. NAV_ITEMS 재구성

### Before (7개 상위메뉴)
```
About(필기/실기/시험안내) | 과목별학습하기 | 학습요약 | 문제풀기 | 대시보드 | 북마크 | 오답노트
```

### After (4개 그룹)
```
시험안내(필기안내/실기안내/시험일정·개요) | 학습하기(과목별학습/학습요약/실기기출해설) | 문제풀기(필기CBT/실기연습) | 나의학습(대시보드/북마크/오답노트)
```

### 변경 이유
- "About" → "시험안내" (한글, 직관적)
- "나의학습" 그룹으로 개인 영역 명확화
- 7개 → 4개 정리 (깔끔한 네비게이션)

## 3. Home 페이지 개인화

### 비로그인 상태
- 기존 랜딩 페이지 유지 (Hero + Subjects + Features + Stats + CTA)

### 로그인 상태 (새 개인화 홈)
- **인사말**: "안녕하세요, {name}님!"
- **빠른 학습 카드 (4개 그리드)**: 필기CBT, 실기연습, 오답복습, 학습요약
- **간단 학습 현황**: 최근 필기/실기 점수, 총 시험 횟수 (Supabase 쿼리)
- **하단 CTA**: 대시보드 상세 분석 링크

### 구현 패턴
```tsx
const { user } = useAuth();
{user ? <LoggedInHome user={user} /> : <LandingHome />}
```
