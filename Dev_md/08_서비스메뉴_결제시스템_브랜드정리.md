## 5. PortOne 결제 플로우

```
1. /about/pricing → 요금제 선택 → /checkout
2. /checkout → 이용권 추가 → 주문자 정보 입력 → 결제 버튼
3. requestPayment() → PortOne V1 SDK (KG이니시스)
4. 결제 성공 → forjob_orders INSERT (paid)
5. SubscriptionContext.refresh() → hasAccess 갱신
6. /confirmation → 결제 완료 확인
```
