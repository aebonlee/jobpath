## 개요

`/info` (시험안내) 페이지의 디자인을 전면 재설계했습니다.
기존에는 CSS 클래스가 대부분 정의되지 않아 스타일 없이 렌더링되는 상태였습니다.

---

## 변경 전 문제점

- `exam-info-table`, `exam-info-subjects`, `exam-info-subject`, `info-box`, `exam-info-links` 등 클래스가 CSS에 미정의
- `exam-info-section` 클래스는 있으나 패딩 값이 부적절
- 스타일 없이 일반 HTML로만 렌더링

## CSS 클래스 목록

### 레이아웃
- `.info-page` - 페이지 컨테이너 (padding-bottom)
- `.info-card` - 카드 섹션 (bg, border, radius, padding)
- `.info-card-header` - 아이콘 + 제목
- `.info-card-icon` - 6가지 색상 변형 (overview, pilgi, silgi, subjects, schedule, links)
- `.info-grid-2` - 필기/실기 2열 그리��

### 하이라이트
- `.info-highlight-row` - 3열 그리드
- `.info-highlight` - 개별 하이라이트 박스
- `.info-highlight-label` / `.info-highlight-value`

### 테이블
- `.info-table` - 기본 테이블 (th 좌��, td 우측)
- `.info-table.schedule` - 일정 테이블 (thead 배경)
- `.info-round-badge` / `.info-round-badge.silgi` - 필기/실기 뱃지

### 과목
- `.info-subjects-grid` - 세로 목록
- `.info-subject-item` - 카드 (CSS 변수 `--accent`로 색상)
- `.info-subject-num` - 원형 번호
- `.info-subject-link` - 우측 화살표 링크

### 링크
- `.info-links-grid` - 3열 그리드
- `.info-link-card` - 링크 카드
- `.info-link-icon` - 좌측 아이콘
- `.info-link-ext` - 외부링크 아이콘
