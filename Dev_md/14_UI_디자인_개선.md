## 1. 학습 기능 카드 하이라이트

### 변경 내용
- 6개 학습 기능 카드 중 3개(CBT 모의시험, 합격 예측, 실기 연습)에 연한 배경색 적용
- 사이트 핵심 기능을 시각적으로 강조

### 구현
- `src/pages/Home.tsx`: `highlight: true` 속성 추가
- `src/styles/home.css`: `.home-feature-card--highlight` 클래스
  - 라이트: `rgba(27, 58, 107, 0.04)` 배경 + `rgba(27, 58, 107, 0.12)` 테두리
  - 다크: `rgba(61, 109, 181, 0.08)` 배경 + `rgba(61, 109, 181, 0.2)` 테두리

## 2. 상단 메뉴 — 글래스모피즘 스타일

### 변경 내용
- 기존 불투명 컬러 배경 → ai-hub 프로젝트 참고 글래스모피즘 디자인
- 반투명 배경 + `backdrop-filter: blur` 효과

### 구현 상세
- **기본 상태**: `rgba(255, 255, 255, 0.92)` + `blur(16px)`
- **스크롤 시**: `rgba(255, 255, 255, 0.98)` + 그림자 강화
- **다크모드**: `rgba(17, 24, 39, 0.92)` / 스크롤 `rgba(17, 24, 39, 0.98)`

### 변경된 요소
| 요소 | 이전 | 이후 |
|------|------|------|
| 네비바 배경 | 불투명 컬러 (`#3B3B3B`) | 반투명 흰색 + blur |
| 텍스트 색상 | 흰색 계열 | `var(--text-primary)` (어두운 색) |
| 로고 색상 | 흰색 | 프라이머리 블루 + 텍스트 |
| hover 효과 | 배경 밝아짐 | 밑줄 애니메이션 (scaleX) |
| 로그인 버튼 | 투명 테두리 | 파란 pill 버튼 |
| 로그아웃 버튼 | 흰색 테두리 | 회색 테두리 + hover 블루 |
| 관리자 링크 | 골드(흰배경용) | 앰버(밝은배경용) / 다크모드 골드 |

### 파일
- `src/styles/navbar.css` — 전체 네비바 스타일
- `src/styles/dark-mode.css` — 다크모드 네비바 오버라이드
