## 콘텐츠 특징

1. **시험 출제 빈도 반영:** 실제 기출문제에서 자주 나오는 이론과 개념 위주로 구성
2. **표(Table) 형식 적극 활용:** 비교, 분류, 정리가 필요한 내용은 HTML 표로 정리
3. **공식박스(formula-box):** 계산 문제 관련 공식은 별도 박스로 강조
4. **예제 포함:** 실기 계산 문제에 구체적 풀이 예제 수록
5. **핵심 키워드 강조:** 시험 답안에 반드시 포함해야 할 키워드를 `<strong>` 태그로 강조
