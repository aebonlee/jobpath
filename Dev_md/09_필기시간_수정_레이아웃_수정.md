## 1. 필기시험 시간 수정 (31e261a)

### 문제

필기시험 시간이 6곳에서 **150분**으로 잘못 표시되고 있었음.
실제 필기시험은 **100문항 100분**, 실기시험이 150분.

### 수정 내역

| 파일 | 위치 | 변경 |
|------|------|------|
| `src/config/site.js` | `EXAM_CONFIG.pilgi.timeLimit` | `150 * 60` → `100 * 60` |
| `src/pages/info/ExamInfo.tsx` | 필기시험 테이블 | "150분" → "100분 (1시간 40분)" |
| `src/pages/practice/PracticeHome.tsx` | 필기 CBT 뱃지 | "150분" → "100분" |
| `src/pages/pilgi/PilgiHome.tsx` | 시험 모드 설명 | "150분 내에" → "100분 내에" |
| `src/pages/pilgi/ExamSelect.tsx` | 안내 텍스트 | "제한시간 150분" → "제한시간 100분" |
| `src/pages/Home.tsx` | 통계 섹션 | "150분" → "100분 (필기시험)" |

### 정확한 시험 정보

| 구분 | 문항수 | 시험시간 | 합격기준 |
|------|--------|---------|---------|
| 필기 | 100문항 (5과목×20) | **100분** | 평균 60점 + 과목당 40점 |
| 실기 | 주관식 | **150분** | 60점 이상 |

## 2. 푸터 하단 고정 (106332e)

### 문제

콘텐츠가 적은 페이지에서 푸터가 화면 중간에 떠 있었음.

### 수정

`App.tsx`의 최상위 Fragment `<>` → `<div className="app-layout">`으로 변경.

```css
/* base.css */
.app-layout {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}
#main-content {
    flex: 1;
}
```

## 3. 필기시험 레이아웃 수정 (2b22eb5)

### 문제

필기시험 문항이 오른쪽 **약 150px 폭**으로만 표시되는 레이아웃 버그.

### 원인

`.exam-container`에 `grid-template-columns: 1fr 280px` 그리드가 적용되어 있어서,
자식 요소 배치가 잘못됨:

```
.exam-container (grid: 1fr 280px)
  ├─ .exam-header    → 1fr 컬럼 (정상)
  ├─ .exam-body      → 280px 컬럼 (문제!)
  │   ├─ .exam-main   → 280px 안에 갇힘
  │   └─ .exam-sidebar → 280px 안에 갇힘
  └─ Modal
```

### 수정

2컬럼 그리드를 `.exam-container` → `.exam-body`로 이동:

```css
/* Before */
.exam-container {
    display: grid;
    grid-template-columns: 1fr 280px;
}

/* After */
.exam-container {
    min-height: calc(100vh - var(--nav-height));
    padding-top: var(--nav-height);
}
.exam-body {
    display: grid;
    grid-template-columns: 1fr 280px;
}
```

반응형 CSS도 `.exam-container` → `.exam-body`로 수정 (2곳: 태블릿 + 프린트).

### 수정 후 구조

```
.exam-container (block)
  ├─ .exam-header (전체 폭)
  ├─ .exam-body (grid: 1fr 280px)
  │   ├─ .exam-main     → 1fr (문제 카드, 전체 폭)
  │   └─ .exam-sidebar   → 280px (문제 네비게이션)
  └─ Modal
```
