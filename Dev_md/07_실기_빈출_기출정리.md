## 데이터 구조 (`silgiFrequent.js`)

### 문항 구조

```javascript
{
  num: 1,                    // 문제 번호 (1~177)
  category: 'counseling',    // 과목 분류
  title: '문제 제목',         // 출제 주제
  frequency: '★★★★★',      // 빈출도 (5단계)
  keyword: '핵심 키워드',     // 검색용 키워드
  answer: '<p>HTML 해설</p>' // HTML 형식 모범답안
}
```

### 과목별 분류

| 카테고리 | 과목명 | 색상 | 문항수 |
|---------|--------|------|--------|
| counseling | 직업상담학 | #3B82F6 | 42 |
| psychology | 직업심리학 | #8B5CF6 | 38 |
| jobinfo | 직업정보론 | #10B981 | 35 |
| labor_market | 노동시장론 | #F59E0B | 32 |
| labor_law | 노동관계법규 | #EF4444 | 30 |

### 헬퍼 함수

- `getQuestionsByCategory(cat)` — 카테고리별 필터링
- `getTotalCount()` — 전체 문항수 반환
- `getCategoryCounts()` — 카테고리별 문항수 객체

## 페이지 구현 (`SilgiFrequent.tsx`)

### 기능

| 기능 | 설명 |
|------|------|
| 카테고리 필터 | 전체 / 5개 과목 컬러 버튼, 문항수 뱃지 |
| 검색 | 제목 + 키워드 실시간 검색, 클리어 버튼 |
| 아코디언 카드 | 클릭 시 해설 펼침/접기, 빈출도 별표시 |
| 전체 펼치기/접기 | 전체 확장/축소 토글 버튼 |
| 네비게이션 | /silgi, /silgi/past 연결 링크 |

### UI 컴포넌트

```
FrequentCard
├─ 카테고리 뱃지 (색상 코딩)
├─ 문제 번호 + 제목
├─ 빈출도 (★)
├─ 키워드 태그
└─ 해설 (아코디언 펼침 시 dangerouslySetInnerHTML)
```
