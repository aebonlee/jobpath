# ExamResult 시험결과 페이지 전면 리디자인

> 작성일: 2026-04-06
> 작업: 시험 결과 페이지 디자인 강화, 과락 과목 가독성 개선

---

## 1. 개요

기존 시험 결과 페이지는 CSS 미정의로 기본 스타일만 적용되어 있었고,
과락 과목이 `<span>` 나열로 띄어쓰기 없이 표시되어 가독성이 떨어졌음.

**Before:**
```
과락 과목:직업상담학 (5점)직업심리학 (0점)직업정보론 (0점)
```

**After:** 각 과목이 개별 카드로 분리, 아이콘/색상/부족점수 표시

---

## 2. 페이지 구조

```
┌──────────────────────────────────────────────────┐
│  히어로 배너 (합격=초록/불합격=빨강 그래디언트)       │
│  아이콘 + 합격/불합격 + 메시지 + 총점               │
├──────────────────────────────────────────────────┤
│  합격기준 체크 카드 2열                              │
│  ✅ 평균 60점 이상: 72점 — 충족                     │
│  ❌ 과목별 40점 이상: 2과목 과락                     │
├──────────────────────────────────────────────────┤
│  4열 통계: 정답수 / 정답률 / 소요시간 / 등급          │
├──────────────────────────────────────────────────┤
│  과락 과목 알림 (있을 경우)                          │
│  ┌ 직업상담학  5점  (-35점 부족) ┐                  │
│  └ 노동관계법규  0점  (-40점 부족) ┘                │
├──────────────────────────────────────────────────┤
│  과목별 성적 (5과목 바차트 + 40점 기준선)             │
├──────────────────────────────────────────────────┤
│  과목별 분석                                        │
│  [레이더차트]  [최고과목/보강과목 카드 + 학습팁]      │
├──────────────────────────────────────────────────┤
│  3열 액션: 오답복습 / 다시시험 / 대시보드             │
└──────────────────────────────────────────────────┘
```

## 3. CSS 클래스 체계 (`er-*` prefix)

### 히어로 배너
| 클래스 | 설명 |
|--------|------|
| `.er-hero` | 배너 래퍼, `position: relative; overflow: hidden` |
| `.er-hero--pass` | 초록 그래디언트 `#059669 → #10B981 → #34D399` |
| `.er-hero--fail` | 빨강 그래디언트 `#B91C1C → #DC2626 → #F87171` |
| `.er-hero-deco` | 장식 원 3개 (흰색 반투명) |
| `.er-hero-icon` | 80px 원형, `backdrop-filter: blur`, 트로피/화살표 아이콘 |
| `.er-hero-badge` | 합격/불합격 텍스트 (28px, 900 weight) |
| `.er-hero-score` | 총점 (64px, 900 weight) |

| `.er-criteria--ok` | 초록 보더 + 체크 아이콘 |
| `.er-criteria--no` | 빨강 보더 + X 아이콘 |

## 4. 로직 변경

### 기존
```tsx
const { isPass, failedSubjects } = checkPass(subjectScores, session.score_total);
```

### 변경
```tsx
const { isPass, failedSubjects, avgPass, allSubjectsPass } = checkPass(...);
const correctRate = Math.round((session.correct_count / questions.length) * 100);
const sortedSubjects = [...SUBJECTS].sort((a, b) => scores[a] - scores[b]);
const bestSubject = [...SUBJECTS].sort((a, b) => scores[b] - scores[a])[0];
```

- `avgPass` / `allSubjectsPass` 활용하여 합격기준 체크 카드 2개 표시
- `correctRate` 정답률 계산
- `sortedSubjects` 점수순 정렬 → 최고/최저 과목 분석

## 8. 해결된 이슈

### 과락 과목 띄어쓰기 문제
**Before:** `과락 과목:직업상담학 (5점)직업심리학 (0점)직업정보론 (0점)`
- `<span>` 인라인 나열, 구분자/간격 없음

**After:** 각 과목이 `.er-fail-item` 개별 카드로 표시
- 아이콘 + 과목명 + 점수 + 부족점수 뱃지
- 한 줄에 한 과목, 시각적으로 완전히 분리
