# JobPath

직업상담사 2급 시험 준비 플랫폼 — 필기·실기 CBT 학습, 오답노트, 합격 예측

## Features

- 필기 CBT 모의시험 (100문항, 100분)
- 실기 연습 및 기출해설
- 과목별 학습 및 요약
- 오답노트 & 북마크
- 학습 대시보드 & 합격 예측
- 구독형 이용권 (30일/90일/평생)

## Tech Stack

- React 19 + Vite 8 + TypeScript
- Supabase (Auth, Database)
- PortOne (결제)
- GitHub Pages

## Development

```bash
npm install
npm run dev      # http://localhost:5190
npm run build
npm run deploy   # gh-pages
```
